## This page contains the codes and data to reproduce the results in the paper titled "Thermally inflated accretors in post-mass transfer binaries: Abell 35 and its class revisited" (Bhattacharjee+, 2026)

For all the Jupyter notebooks to work, the following needs to be performed

1) Make a directory named 'grid' and download the PHOENIX stellar models from "https://archive.stsci.edu/hlsps/reference-atlases/cdbs/grid/phoenix/" and store the contents in the directory
2) Run the .sh script inside folder 'bosz_models_manual' to download the BOSZ spectral models
